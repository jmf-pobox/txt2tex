\section*{Phase 11 d : Lambda Expressions}

\bigskip

Lambda expressions provide a concise way to define anonymous functions.

\bigskip

(a)
\par
\vspace{11pt}
\bigskip

Simple lambda expression

\bigskip

\noindent
$\lambda x : \nat @ x^2$

\medskip

(b)
\par
\vspace{11pt}
\bigskip

Multi-variable lambda

\bigskip

\noindent
$\lambda x, y : \nat @ x \land y$

\medskip

(c)
\par
\vspace{11pt}
\bigskip

Lambda with comparison

\bigskip

\noindent
$\lambda x : \nat @ x > 0$

\medskip

(d)
\par
\vspace{11pt}
\bigskip

Nested lambda expression

\bigskip

\noindent
$\lambda x : X @ \lambda y : Y @ x \land y$

\medskip

(e)
\par
\vspace{11pt}
\bigskip

Lambda with quantifier in body

\bigskip

\noindent
$\lambda x : X @ \forall y : Y @ x \lor y$

\medskip

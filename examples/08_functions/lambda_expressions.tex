\section*{Phase 11 d : Lambda Expressions}

\noindent Lambda expressions provide a concise way to define
anonymous functions.

\bigskip

\medskip

\subsection*{(a)}

\noindent Simple lambda expression

\bigskip

\noindent
$\lambda x : \nat @ x \bsup 2 \esup$

\medskip

\subsection*{(b)}

\noindent Multi-variable lambda

\bigskip

\noindent
$\lambda x, y : \nat @ x \land y$

\medskip

\subsection*{(c)}

\noindent Lambda with comparison

\bigskip

\noindent
$\lambda x : \nat @ x > 0$

\medskip

\subsection*{(d)}

\noindent Nested lambda expression

\bigskip

\noindent
$\lambda x : X @ \lambda y : Y @ x \land y$

\medskip

\subsection*{(e)}

\noindent Lambda with quantifier in body

\bigskip

\noindent
$\lambda x : X @ \forall y : Y @ x \lor y$

\section*{Identifier Decoration in Z}

\noindent Z notation uses decorated identifiers to distinguish input, output, and after-state variables. A prime (') marks the after-state; a question mark (?) marks an input; an exclamation mark (!) marks an output.

\bigskip

\begin{schema}{StateOp}
count : \nat \\
count' : \nat \\
in? : \nat \\
out! : \nat
\where
count' = count + in? \\
out! = count
\end{schema}

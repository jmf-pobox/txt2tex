\section*{Phase 20 Demo : bigcup ( Distributed Union )}

\bigskip
\noindent
\textbf{Example 1 : Basic bigcup}

\medskip

\bigskip

bigcup S is the union of all sets in the set of sets S:

\bigskip

\begin{zed}[X]\end{zed}

$abbrev$

S == \power~\power~X

\begin{axdef}
allElements : S \fun \power~X
\where
\forall s \colon S \bullet allElements(s) = \bigcup s
\end{axdef}

\bigskip
\noindent
\textbf{Example 2 : bigcup with ran}

\medskip

\bigskip

Combining bigcup with ran to get all elements from a relation's range:

\bigskip

\begin{zed}[UserId, SongId]\end{zed}

\begin{axdef}
loved : UserId \pfun \finset~SongId
allLovedSongs : \finset~SongId
\where
allLovedSongs = \bigcup \ran loved
\end{axdef}

\bigskip
\noindent
\textbf{Example 3 : Set difference with bigcup ( Solution 50 a )}

\medskip

\bigskip

Users who are not subscribers of any playlist:

\bigskip

\begin{zed}[PlaylistId, UserId]\end{zed}

\begin{axdef}
users : \finset~UserId
playlistSubscribers : PlaylistId \pfun \finset_1~UserId
nonSubscribers : \finset~UserId
\where
nonSubscribers = users \setminus \bigcup \ran playlistSubscribers
\end{axdef}

\bigskip
\noindent
\textbf{Example 4 : Nested bigcup}

\medskip

\bigskip

bigcup can be nested for multi-level set structures:

\bigskip

\begin{zed}[N]\end{zed}

\begin{axdef}
nested : \power~\power~\power~N
\where
\bigcup \bigcup nested
subseteq
N
\end{axdef}

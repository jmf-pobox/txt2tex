\section*{Recursive Function Examples}

\begin{zed}[Entry]\end{zed}

\bigskip

This file demonstrates recursive functions on sequences.

\bigskip

\bigskip
\noindent
\textbf{Example 1 : Simple recursive function}

\medskip

\begin{axdef}
length : \seq Entry \fun \nat
\where
length(\langle \rangle) = 0 \\
\forall x : Entry @ \forall s : \seq Entry @ length(\langle x \rangle \cat s) = 1 + length(s)
\end{axdef}

\bigskip
\noindent
\textbf{Example 2 : Identity function on sequences}

\medskip

\begin{axdef}
identity : \seq Entry \fun \seq Entry
\where
identity(\langle \rangle) = \langle \rangle \\
\forall x : Entry @ \forall s : \seq Entry @ identity(\langle x \rangle \cat s) = \langle x \rangle \cat identity(s)
\end{axdef}

\bigskip
\noindent
\textbf{Example 3 : Sequence doubling}

\medskip

\begin{axdef}
duplicate : \seq \nat \fun \seq \nat
\where
duplicate(\langle \rangle) = \langle \rangle \\
\forall n : \nat @ \forall s : \seq \nat @ duplicate(\langle n \rangle \cat s) = \langle n, n \rangle \cat duplicate(s)
\end{axdef}

\bigskip

These examples show pattern matching on sequences with base cases and recursive cases.

\section*{Phase 18 Demo : Partial Function Operators}

\bigskip
\noindent
\textbf{Example 1 : -|> operator ( partial injection )}

\medskip

\bigskip

Basic function type with -|> operator:

\bigskip

\begin{axdef}
f : N \pinj N
\where
\dom f = \{1, 2, 3\}
f(1) = 10
f(2) = 20
f(3) = 30
\end{axdef}

\bigskip
\noindent
\textbf{Example 2 : +-> operator ( partial function )}

\medskip

\bigskip

Basic function type with +-> operator:

\bigskip

\begin{axdef}
g : N \pfun N
\where
\dom g = \{1, 2\}
g(1) = 100
g(2) = 200
\end{axdef}

\bigskip
\noindent
\textbf{Example 3 : Complex types with -|>}

\medskip

\bigskip

Using -|> with custom types:

\bigskip

\begin{zed}[Year, Entry]\end{zed}

$abbrev$

Table == \seq~Entry

\begin{axdef}
records : Year \pinj Table
\where
\dom records = \{1993, 1994, 1995\}
\end{axdef}

\medskip

\bigskip

=== Phase 18 Demo: Partial Function Operators ===

** Example 1: Partial injection operator **

TEXT: Basic function type with partial injection (-|>) operator:

axdef
  f : N -|> N
where
  dom(f) = {1, 2, 3}
  f(1) = 10
  f(2) = 20
  f(3) = 30
end

** Example 2: Partial function operator **

TEXT: Basic function type with partial function (+->) operator:

axdef
  g : N +-> N
where
  dom(g) = {1, 2}
  g(1) = 100
  g(2) = 200
end

** Example 3: Complex types with partial injection **

TEXT: Using partial injection (-|>) with custom types:

Year == N

given Entry

Table == seq(Entry)

axdef
  records : Year -|> Table
where
  dom(records) = {1993, 1994, 1995}
end

** Example 4: Mixed function type operators **

TEXT: Different function type operators:

=== Schema Examples ===

** Example 1: Simple Schema **

schema Counter
  count : N
where
  count >= 0
end

** Example 2: Schema with Multiple Fields **

schema Point
  x : Z
  y : Z
end

** Example 3: Schema with Constraints **

schema BoundedPoint
  x : N
  y : N
where
  x <= 100
  y <= 100
end

** Example 4: Schema with Given Types **

given Person

schema PersonData
  name : Person
  age : N
where
  age >= 0
  age <= 150
end

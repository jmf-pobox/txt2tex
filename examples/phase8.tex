\section*{Phase 8 : Set Comprehension}

\bigskip
\noindent
\textbf{Example 1 : Set by Predicate}

\medskip

(a)
\par
\vspace{11pt}
\bigskip

Basic set comprehension with domain

\bigskip

$\{ x \colon N \mid x > 0 \}$

\medskip

(b)
\par
\vspace{11pt}
\bigskip

Multi variable set

\bigskip

$\{ x, y \colon N \mid x = y \}$

\medskip

(c)
\par
\vspace{11pt}
\bigskip

Set without domain

\bigskip

$\{ x \mid x \in A \}$

\medskip

\bigskip
\noindent
\textbf{Example 2 : Set by Expression}

\medskip

(a)
\par
\vspace{11pt}
\bigskip

Set comprehension with expression part

\bigskip

$\{ x \colon N \mid x > 0 \bullet x^2 \}$

\medskip

(b)
\par
\vspace{11pt}
\bigskip

Combining set comprehension with quantifier

\bigskip

$\forall x \colon N \bullet x \in \{ y \colon N \mid y > 0 \}$

\medskip

\bigskip
\noindent
\textbf{Example 3 : Complex Examples}

\medskip

(a)
\par
\vspace{11pt}
\bigskip

Set comprehension in equivalence

\bigskip

$\{ x \colon N \mid x > 0 \} = \{ y \colon N \mid y > 0 \}$

\medskip

(b)
\par
\vspace{11pt}
\bigskip

Set comprehension with multiple conditions

\bigskip

$\{ x \colon N \mid x > 0 \land x \leq 10 \}$

\medskip

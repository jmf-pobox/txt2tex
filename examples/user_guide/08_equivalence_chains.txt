// Example: Equivalence Chains with Justifications
// Demonstrates EQUIV: blocks with free-form justifications
// From USER_GUIDE.md - Propositional Logic section

EQUIV:
p land q
<=> q land p [commutative]
<=> q land p [idempotent]

EQUIV:
lnot (p land q)
<=> (lnot p) lor (lnot q) [De Morgan]
<=> (lnot p) lor (lnot q) [idempotence]
<=> (lnot p) lor (lnot q) lor (lnot r) [associativity]
<=> true [excluded middle]

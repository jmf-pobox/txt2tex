// Example: Proof of Distributivity with Siblings
// Demonstrates sibling premises (::) in case analysis
// From USER_GUIDE.md - Proof Trees section

PROOF:
p land (q lor r) => (p land q) lor (p land r) [=> intro from 1]
  [1] p land (q lor r) [assumption]
      p [land elim 1]
      q lor r [land elim 2]
      (p land q) lor (p land r) [lor elim]
        case q:
          :: p [from above]
          :: q [from case]
          p land q [land intro]
          (p land q) lor (p land r) [lor intro 1]
        case r:
          :: p [from above]
          :: r [from case]
          p land r [land intro]
          (p land q) lor (p land r) [lor intro 2]

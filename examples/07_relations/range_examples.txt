=== Testing Range Operator ===

** Test 1: Simple Numeric Range **
1..10

** Test 2: Range with Identifiers **
1993..current

** Test 3: Range in Set Comprehension **
{ x : 1..20 | x mod 2 = 0 }

** Test 4: Range with Expressions **
a + 1..b + 2

** Test 5: Range in Schema **
schema ValidYear
  year : 1900..2100
where
  year >= 1900
end

** Test 6: Union of Ranges **
1..5 union 10..15

** Test 7: Range with Tuple Projections **
p.1..p.2

** Test 8: Complex Set Comprehension **
{ x : 1..10 | x > 3 . x^2 }

# Lecture 1: Introduction and Propositions

This directory contains examples for Lecture 1, covering propositional logic.

## Topics Covered

- Basic logical operators: `and`, `or`, `not`, `=>`, `<=>`
- Operator precedence
- Truth tables
- Equivalence chains with justifications
- Complex formulas

## Key Operators

```
not p            →  ¬p          (negation)
p and q          →  p ∧ q       (conjunction)
p or q           →  p ∨ q       (disjunction)
p => q           →  p ⇒ q       (implication)
p <=> q          →  p ⇔ q       (equivalence)
```

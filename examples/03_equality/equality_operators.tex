\section*{Equality Operators}

\bigskip
\noindent
\textbf{Example 1 : Basic Equality}

\medskip

\bigskip

The equality operator compares values:

\bigskip

\noindent
$x = y$


\noindent
$x \neq y$


\bigskip
\noindent
\textbf{Example 2 : Equality in Predicates}

\medskip

\bigskip

Equality is commonly used in quantified predicates:

\bigskip

\noindent
$\forall x \colon \mathbb{N} \bullet x = x$


\noindent
$\exists y \colon \mathbb{Z} \bullet y = 0$


\bigskip
\noindent
\textbf{Example 3 : Equality in Set Comprehension}

\medskip

\bigskip

Using equality to define sets:

\bigskip

\noindent
$\{ z \colon \mathbb{Z} \mid z = 10 \}$


\noindent
$\{ n \colon \mathbb{N} \mid n = n \}$


\bigskip
\noindent
\textbf{Example 4 : Equality with Expressions}

\medskip

\bigskip

Equality can compare complex expressions:

\bigskip

\noindent
$\forall x, y \colon \mathbb{N} \bullet x + y = y + x$


\noindent
$\exists a, b \colon \mathbb{Z} \bullet a * b = 0$

\section*{Phase 8 : Set Comprehension}

\bigskip
\noindent
\textbf{Example 1 : Set by Predicate}

\medskip

(a)
\par
\vspace{11pt}
\bigskip

Basic set comprehension with domain

\bigskip

\noindent
$\{ x : \nat | x > 0 \}$

\medskip

(b)
\par
\vspace{11pt}
\bigskip

Multi variable set

\bigskip

\noindent
$\{ x, y : \nat | x = y \}$

\medskip

(c)
\par
\vspace{11pt}
\bigskip

Set without domain

\bigskip

\noindent
$\{ x | x \in A \}$

\medskip

\bigskip
\noindent
\textbf{Example 2 : Set by Expression}

\medskip

(a)
\par
\vspace{11pt}
\bigskip

Set comprehension with expression part

\bigskip

\noindent
$\{ x : \nat | x > 0 @ x^2 \}$

\medskip

(b)
\par
\vspace{11pt}
\bigskip

Combining set comprehension with quantifier

\bigskip

\noindent
$\forall x : \nat @ x \in \{ y : \nat | y > 0 \}$

\medskip

\bigskip
\noindent
\textbf{Example 3 : Complex Examples}

\medskip

(a)
\par
\vspace{11pt}
\bigskip

Set comprehension in equivalence

\bigskip

\noindent
$\{ x : \nat | x > 0 \} = \{ y : \nat | y > 0 \}$

\medskip

(b)
\par
\vspace{11pt}
\bigskip

Set comprehension with multiple conditions

\bigskip

\noindent
$\{ x : \nat | x > 0 \land x \leq 10 \}$

\medskip

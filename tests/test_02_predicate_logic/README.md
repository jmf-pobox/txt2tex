# Test 02: Predicate Logic

Tests for quantifiers and predicate logic (Lecture 2).

## Test Coverage

- Universal quantification: `forall x : T | predicate`
- Existential quantification: `exists x : T | predicate`
- Unique existence: `exists1 x : T | predicate`
- Quantifier nesting and complex predicates
- Type declarations in quantifiers
- Complex types (relations, functions) in bindings
